# My LLM from scrach

## Installation

Update [this `mise` setting](./mise.toml), then do following:

```bash
mise trust
mise install

uv sync
```

## How to edit

Open `marimo` notebook like this:

```bash
marimo edit --watch ./src/my-llm/ch06.py
```
